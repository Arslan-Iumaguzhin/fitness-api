Ты — бэкенд-разработчик API для платформы, помогающей пользователям отслеживать физическую активность.
Необходимо создать полноценный сервис на Flask, который выполняет следующие функции:

Основные функции:

Создает пользователя
Проверяет корректность e-mail и номера телефона (например, с помощью регулярных выражений).
Хранит имя, возраст и цель пользователя (например, «похудеть», «набрать массу», «поддерживать форму»).

Выдает данные по конкретному пользователю
(например, статистику тренировок, количество выполненных упражнений и т. д.).

Генерирует тренировку дня
Создает случайный набор упражнений (из заранее заданного списка), указывая количество подходов и повторений.
Пользователь выполняет тренировку и отправляет результат (например, количество повторений). За выполненную тренировку начисляются баллы.

Создает вопрос о здоровом образе жизни
(два типа вопросов: с вариантами ответов и с открытым ответом).
За правильный ответ начисляются баллы.

Выдает историю тренировок и ответов на вопросы пользователя

Генерирует таблицу или график прогресса пользователя по набранным баллам

Допущения:

Все объекты (пользователи, тренировки, вопросы) можно хранить в runtime (например, в списках или словарях).

Проверку существования e-mail и номера телефона можно реализовать через регулярные выражения, без подключения внешних библиотек.

Формирование графика или таблицы можно сделать в виде имитации данных (без фронтенда).

Необходимо:

Код должен быть отформатирован (например, при помощи black).

Проект должен содержать четкую структуру (например, app.py, routes/, models/, utils/).

Все основные функции должны быть доступны через REST API (GET, POST, PUT, DELETE, если нужно).

Примерный стек:

Python 3.10+

Flask

Regex (для валидации)

Random (для генерации тренировок и вопросов)

Matplotlib или аналог (для генерации графика, опционально)
